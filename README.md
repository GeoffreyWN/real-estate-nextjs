

A real estate app created with NextJs, Chakra-UI

## Getting Started

### Steps
1. clone the repo
2. cd into repo
3. npm install
4. update env
4. npm run dev

Open [http://localhost:3000](http://localhost:3000) with your 
browser to see the result.

Enjoy :-)
